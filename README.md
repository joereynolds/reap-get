#reap-get

A package manager for VST audio plugins :D

# Usage

###Clone me

```
    git clone https://github.com/joereynolds/reap-get.git
    cd reap-get
```

### Use me 

run as administrator in Windows...sigh

##### View all packages
```
    python reap-get.py --view 
```
##### Download a package
```
    python reap-get.py --install synth1
```
